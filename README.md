# SpendByte Landing Website

This is a premium, modern, startup-style landing page designed for the "SpendByte" Android application.
It is built using pure HTML, CSS, and JS, meaning it is incredibly lightweight and easy to host on any static site provider (GitHub Pages, Netlify, Vercel).

## Features
- Fully responsive mobile-first design
- SEO friendly metadata
- Smooth reveal-on-scroll animations using IntersectionObserver
- Interactive CSS-based screenshots carousel
- Premium dark mode "Fintech" styling with soft gradients and glows

## How to use
1. Add your real app screenshots (or use the pure-CSS placeholders provided)
2. Replace `spendbyte-app.apk` with your real compiled Android bundle
3. Host the directory index.html anywhere!
